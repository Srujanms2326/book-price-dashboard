# 📊 Book Price Dashboard

An interactive data dashboard built using **Python** that scrapes, cleans, and visualizes book pricing data. The application provides real-time insights with filtering, sorting, and graphical representation using **Streamlit**.

---

## 🚀 Live Demo

🔗 [https://book-price-dashboard.streamlit.app](https://book-price-dashboard-gignwreji8zcwpyxaabpa3.streamlit.app/)

---

## 📁 Project Overview

This project extracts book data from a website, processes it, and displays meaningful insights through an interactive dashboard.

### ✨ Features

* 📚 Web scraping using BeautifulSoup
* 🧹 Data cleaning and preprocessing
* 📊 Interactive dashboard using Streamlit
* 🔍 Price filtering using slider
* 📈 Data visualization (bar charts)
* 💰 Currency formatting (₹ display)
* 📉 Statistical insights (min, max, average)

---

## 🛠️ Tech Stack

* **Python**
* **Pandas**
* **BeautifulSoup**
* **Streamlit**
* **Requests**

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Srujanms2326/book-price-dashboard.git
cd book-price-dashboard
```

### 2️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Run the application

```bash
streamlit run dashboard.py
```

---

## 📊 How It Works

1. Data is scraped from a website using BeautifulSoup
2. Raw data is stored in CSV format
3. Data is cleaned using Pandas (removing symbols, handling errors)
4. Streamlit displays the processed data with filters and charts
